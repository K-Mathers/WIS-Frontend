# 👟 WIS Frontend

**WIS Frontend** is a modern web application built with **React + Vite**, feat. UI components with **Ant Design**, data fetching with **TanStack React Query**.

---

## 📋 Stack 

- **React**: v19
- **Build Tool**: Vite
- **Styling**: CSS, Ant Design
- **State Management**: TanStack React Query v5
- **Utils**: React Hook Form, Zod, Framer Motion, Socket.io-client, Axios, Tiptap

---

## 🚀 Quick Start Guide

### 1. Installation

Clone the repository and install dependencies:

```bash
npm install
```

### 2. Running the Application

```bash
# Dev mode
npm run dev

# Production build
npm run build

# Preview build
npm run preview
```

## 📁 Project Structure

- `src/api` - API requests and networking
- `src/assets` - Static assets like images and icons
- `src/components` - Reusable UI components
- `src/const` - Application constants
- `src/context` - React Context providers
- `src/hooks` - Custom React hooks
- `src/pages` - Application pages for routing
- `src/schemas` - Validation schemas (Zod)
- `src/types` - TypeScript interfaces and types
- `src/utils` - Helper functions
